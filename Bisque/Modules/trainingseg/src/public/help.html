I. Input

Match the number

For each subjects, we should have 2 files; One is raw scan and the other is segmented image(ground truth).
If the number of files between Scans and Segmentation are different, Bisque will throw an error.

Naming the files

For the convenience of users, we highly recommend you to put subject's information in your file names like:

Scans : { SUBJECT INFO }.nii.gz

Segmentation : Final_{ SUBJECT INFO }.nii.gz


If you did not put Final_ in front of Segmentations files, module cannot read your files.
